# Okcoin API Ruby Wrapper
Supports REST and Websocket protocols

## Installation

Add this line to your application's Gemfile:

```ruby
gem 'okcoin-ruby'
```

And then execute:

    $ bundle

Or install it yourself as:

    $ gem install okcoin-ruby

## Usage

Use it with REST and Websocket protocols
